## Amortization Schedule
- Built a basic amortization schedule from first principles and visualized loan dynamics (balance, interest, principal) over time.
  
### Features
- Computes level payment using PMT
- Generates full amortization schedule
- Visualizes repayment structure via line chart

### Concepts Covered:
- Time Value of Money
- Annuities
- Recurrence relations
